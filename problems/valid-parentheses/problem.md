# Valid Parentheses

**Difficulty:** Easy

## Problem Description

Given a string s containing just the characters '(', ')', '{', '}', '[' and ']', determine if the input string is valid.

## Examples

### Example 1:

```
Input: s = "()"
Output: true
```

### Example 2:

```
Input: s = "()[]{}"
Output: true
```

## Constraints

- 1 <= s.length <= 10^4
- s consists of parentheses only '()[]{}'
